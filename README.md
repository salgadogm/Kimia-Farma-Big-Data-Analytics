# Performance Analytics Kimia Farma (2020-2023)

Proyek ini adalah bagian dari *Big Data Analytics Internship* di Kimia Farma. Tujuannya adalah untuk menganalisis kinerja bisnis perusahaan menggunakan BigQuery dan Looker Studio.

## Isi Repositori
- **dataset/**: Berisi file mentah CSV (transaksi, produk, cabang, dan inventaris).
- **scripts/**: Berisi file SQL untuk membuat `tabel_analisa`.

## Langkah Pengerjaan
1. Mengimpor dataset asli ke Google BigQuery.
2. Membuat tabel denormalisasi (`tabel_analisa`) dengan SQL untuk menggabungkan data dan menghitung metrik finansial (Net Sales & Net Profit).
3. Melakukan visualisasi data menggunakan Google Looker Studio.
